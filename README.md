# 🛒 Amazon Sales Analysis Dashboard Suite

An interactive Power BI dashboard analyzing Amazon sales performance, customer behavior, revenue trends, and key business KPIs. This project transforms e-commerce transactional data into high-level strategic insights to assist in optimizing retail decisions and performance forecasting.

---

## 📌 Project Overview
In the highly competitive e-commerce sector, unlocking data patterns from sales operations is essential. This repository features a comprehensive data dashboard based on Amazon marketplace datasets. 

By deep-diving into sales distribution, operational execution metrics, item profit margins, and dynamic timeline performance, this project demonstrates robust business intelligence solutions.

---

## 📂 Repository Structure
The files in this repository are organized as follows:

*   **`Logos/`**: Design assets and thematic logos used to elevate the dashboard interface styling.
*   **`Amazon Task 6.pbix`**: The underlying Power BI desktop file containing calculations, relationships, and presentation layers.
*   **`Amazon.csv`**: The primary transactional e-commerce dataset analyzed.
*   **`Amazon Dash.pdf`**: A high-resolution print export of the completed analytical dashboard.
*   **`Amazon Dash.vid.mp4`**: A video walk-through demonstrating interactive features, navigation, and cross-filtering.

## 🚀 Key Features & Insights
*   **Revenue & Margin Tracking**: Comprehensive evaluation of top-line revenue, operational profit figures, and percentage contribution breakdowns.
*   **Customer Behavior Analysis**: Detailed segment matrices mapping consumption frequency and product category preferences.
*   **Timeline Trends**: Granular time-intelligence insights exploring peak seasonal shifts and promotional response cycles.

---

## 🛠️ Tools & Technologies Used
*   **Power BI Desktop**: Data preparation modeling, complex business metrics definition, and structural visualization development.
*   **Power Query**: Structural column modifications, metadata alignment, and data optimization.
*   **DAX (Data Analysis Expressions)**: Dynamic expressions assessing multi-variable sales and date calculations.
