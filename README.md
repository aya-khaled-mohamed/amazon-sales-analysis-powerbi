# Amazon Sales Analysis Dashboard Suite

## Overview

This repository contains a collection of Power BI dashboards designed to analyze Amazon sales performance, customer behavior, revenue trends, product performance, and key business KPIs.

## Project Files

* Amazon Task 6.pbix → Power BI project file
* Amazon.csv → Dataset used for analysis
* Amazon Dash.pdf → Complete dashboard report containing multiple dashboard pages
* Amazon Dash.vid.mp4 → Dashboard walkthrough video
* Logos/ → Visual assets, icons, and logos used in dashboard design

## Dashboard Coverage

The dashboard suite includes:

* Sales Performance Analysis
* Revenue Trend Analysis
* Customer Insights
* Product Performance Monitoring
* KPI Tracking and Reporting

## Tools & Technologies

* Power BI
* Power Query
* DAX
* CSV Data Sources
